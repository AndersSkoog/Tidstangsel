Development setup: 
	Clone the repo, 
	install bun https://bun.sh on your machine, 
	then from the Cloned repo directory, do: 

	> bun install
	> bun run build_client 
	> bun run build_client_test
	
	start production server: 
	> bun run start_prod
	start dev server: 
	> bun run start_dev
